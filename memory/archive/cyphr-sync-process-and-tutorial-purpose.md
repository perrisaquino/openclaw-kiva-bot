## 🎯 TUTORIAL SYSTEM PURPOSE (CORRECTED)

### **PRIMARY FUNCTION: DEVICE CALIBRATION THROUGH GUIDED FIRST RECORDING**

The tutorial system **kills multiple birds with one stone** by combining:

1. **User Onboarding** - Teaching core app functionality
2. **Device Calibration** - Essential for music sync to work properly  
3. **First Success** - Ensuring user gets a working recording immediately
4. **Sync Validation** - Proving the core value proposition works

### **WHY CALIBRATION IS CRITICAL**

**CyphrCam's core value:** Practice with headphones, auto-sync music to recorded movement

**The Problem:** Every device has different audio latency between:
- Headphone audio (what user hears while dancing)
- Video recording audio (what gets captured)
- Different audio devices (wired vs. Bluetooth vs. speakers)

**The Solution:** Calibration process that measures this latency difference for each user's specific setup.

**Tutorial's Role:** Guides user through their FIRST calibration recording, which:
- Teaches them how to use the app
- Calibrates their device automatically  
- Produces a perfectly synced result immediately
- Proves the technology works as promised

## 🔄 THE CYPHR SYNC PROCESS

### **Step 1: Music Library Setup**
- Add songs (upload or Apple Music sync)
- Set start points and loops if desired
- Create playlists for practice sessions

### **Step 2: Recording with Headphones**
- User dances with headphones to their music
- App records video of movement
- App captures reference audio track
- User hears music privately through headphones

### **Step 3: Automatic Sync Detection**
- App analyzes movement timing vs. music timing
- Identifies latency between headphone audio and recorded audio
- Calculates device-specific calibration offset

### **Step 4: Sync Refinement (If Needed)**
- User can fine-tune sync in refinement tool
- Make micro-adjustments for perfect alignment
- Switch audio devices if needed (Bluetooth → speakers)

### **Step 5: Export with Perfect Sync**
- Final video has music perfectly aligned to movement
- No manual editing required
- Ready to share immediately
