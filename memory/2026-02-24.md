- **Service management** - auto-restart if crashed, proper logging
- **Management commands** - easy start/stop/status checking

### Dashboard Integration:
- **Server connection detection** - shows online/offline status
- **Auto-sync every 30 seconds** - keeps data persistent
- **Live weather widget** - displays current conditions in dashboard
- **Activity logging** - all actions logged to both local and server
- **Graceful fallback** - works offline with localStorage if server unavailable

### Setup Files Created:
- `server.js` - main server file (9KB, single file, no heavy frameworks)
- `package.json` - minimal dependencies (just Express)
- `com.missioncontrol.server.plist` - LaunchAgent template
- `install-launchagent.sh` - auto-installer with error handling
- `start.sh` - simple startup script
- `README.md` - quick reference
- `SETUP.md` - comprehensive documentation (5KB)

### Usage:
