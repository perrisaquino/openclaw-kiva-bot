## 🔍 **RELATED NOTES SEARCH PROCESS:**

### **1. Extract Keywords from Reflection:**
- Core concepts mentioned
- Names of people  
- Emotional themes
- Technical terms
- Places/contexts

### **2. Run Vault Search:**
```bash
find "/Users/iamperris/Local Documents/Main Second Brain (Local)/" -name "*.md" -type f -exec grep -l "KEYWORD1\|KEYWORD2\|KEYWORD3" {} \; | head -10
```

### **3. Verify Connections:**
- Read found files to confirm relevance
- Only include connections that actually relate
- Explain WHY each existing note connects

### **4. Structure Result:**
- **Existing Vault Connections** FIRST (minimum 3-5)
- **Potential New Notes** SECOND (for future development)

## ❌ **NEVER DO:**
- Skip the vault search step
- Make up note titles that don't exist
- Put "potential notes" before existing connections
- Use generic explanations without specific relevance
