# Daily Reflection & Quick Capture Skill

Process voice notes and brain dumps into structured Quick Capture notes in Obsidian.

## When to Use
- Processing reflections from the Quick Capture / Daily Reflection Telegram group
- Creating structured notes from voice memos or text dumps
- Filling in daily journal evening check-ins

## Files
- `QUICK-CAPTURE-TEMPLATE-WITH-RULE.md` - The template to follow exactly
- `RELATED-NOTES-RULE.md` - Rules for finding related vault notes
- `cron-jobs/` - Weekly content pipeline cron

## Rules
- Follow the template EXACTLY (emojis, line spaces between checkboxes, conversation note separation)
- Search vault FIRST for related notes, never make up titles
- Preserve raw reflection text exactly as spoken
- Use copywriting skill for headlines (specific, not vague)
- Save to: `/Users/iamperris/Local Documents/Main Second Brain (Local)/Journal/Daily/Quick Captures/`
- Update 📝 QUICK CAPTURES.md feed with new entries
